# Kanban MVP

Single-board Kanban app built with Next.js (App Router, client-rendered).

## Features

- One board with 5 fixed renameable columns
- Cards with title + details
- Drag-and-drop movement across columns
- Add and delete cards
- Seeded dummy data on load

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Test

```bash
npm run test
npm run test:e2e
```
